# CHANGELOG

All notable changes to the "casl2-debug" extension will be documented in this file.

## [1.0.0] - 2025-01-08

### Added
- Initial release
- Debug Adapter Protocol support for CASL2/COMET2
- Step execution (step, step in, step out)
- Breakpoint support at source lines
- Register inspection (PC, FR, GR0-GR7, SP)
- Stack trace viewing
- CASL2 syntax highlighting
- Auto-start of c2c2 DAP server
- Configurable debug server port
- Configurable c2c2 executable path
